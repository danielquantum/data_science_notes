*case sensitive*

Text
```python
print("Hellow World!")
```

Number
```python
print(3)
```

Formatting
```python
name = 'Daniel'
number = 19.500
print(f"Hello {name}, my number is {number:.2f}!")

print("Hello {}, my number is {}".format{name, number})

print("Hello {0}, my number is {1}".format{name, number})

print("Hello %s, my number is %d"%{name, number})
```

| ------ | -------- |
