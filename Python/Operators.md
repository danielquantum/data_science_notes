## Comparison

| Comparison | Symbol | Symbol |
| ---------- | ------ | ------ |
| less       | i < j  | i <= j |
| more       | i > j  | i >= j |
| equal      | i == j |        |
| not equal  | i != j |        |
## Logical operation

| Order | Symbol          | Output |
| ----- | --------------- | ------ |
| NOT   | not 4 < 5       | False  |
| AND   | 4 < 5 and 3 < 2 | False  |
| OR    | 4 < 5 or 3 < 2  | True   |
