## Primitive Data Structure

Type
```python
type("3.0")       # str
```

| type  | casting       | output |
| ----- | ------------- | ------ |
| int   | int(3.9)      | 3      |
|       | round(3.9)    | 4      |
|       | round(3.9, 2) | 4.00   |
| float | float(3)      | 3.0    |
| str   | str(3.9)      | '3.9'  |
| bool  | bool(1)       | True   |
|       | bool(0)       | False  |
| None  | None          | None   |
